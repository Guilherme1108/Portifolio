Projetos Anteriores

Site League of Legends

Este projeto é um site inspirado no League of Legends da Riot Games, desenvolvido como parte de um trabalho avaliativo com o objetivo de praticar conceitos de desenvolvimento web. Foi criado utilizando HTML e Tailwind CSS, com foco em um design moderno e totalmente responsivo, adaptando-se a diferentes tamanhos de tela — desde desktops até dispositivos móveis. O projeto representa uma aplicação prática dos conhecimentos adquiridos sobre estruturação de páginas, estilização com frameworks e boas práticas de desenvolvimento front-end.

API Locadora de Filmes

Este projeto implementa uma API integrada a banco de dados que simula o funcionamento de uma locadora de filmes. Através dele, é possível cadastrar, consultar, atualizar e remover filmes, gerenciando informações como título, gênero e disponibilidade. A aplicação está sendo desenvolvida com foco em práticas reais de back-end e persistência de dados, proporcionando uma interfaceprogramática e organizada para trabalhar com operações sobre registros de filmes.
